# MatkulTI

MatKul adalah aplikasi web semantik untuk pencarian Informasi tentang Mata Kuliah Teknik Informatika Unpad menggunakan RDF dan SPARQL

Aplikasi ini dibuat menggunakan Symfony 5 PHP Framework
